# Derm AI: Intelligent Skin Cancer Detection Engine

A full-stack medical diagnostics prototype designed to optimize early-stage epidermal lesion screening and localized clinical specialist routing.

## Key Features
* AI Diagnostic Processing:Deep learning analysis utilizing a DenseNet121 CNN architecture trained on the HAM10000 dataset to classify 7 lesion classifications.
* gDynamic Localization Routing: Smart backend modules that query a localized provider database to surface medical specialists if high-risk targets are flagged.

## Tech Stack
* Deep Learning Framework: TensorFlow, Keras, NumPy, Scikit-learn
* Backend Framework: Python, Flask, SQLite3
* Frontend UI Dashboard: HTML5, Tailwind CSS, Chart.js
